Для открытия консоли в папке PlanetJson: Зайди в эту папку, кликни чуть левее её имени и напиши cmd

После изменения любого файла пиши в консоль:
git pull
git add .
git commit -m "Changed texture имя изменённой текстуры"
git push

S - save
L - load
C - calculate (в консоль выводит всё чё надо)
U - жёсткий апдейт (удаляет все постройки и ландшафт, изменяет размерность планеты)
X - мягкий апдейт (изменяет только название)
G - генерация ландшафта
